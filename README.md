# Galeria de Supercarros 3D

Este repositório contém a estrutura de uma galeria interativa de supercarros, utilizando HTML5 e variáveis de estilo CSS para posicionamento dinâmico.

## Estrutura do Código

O projeto utiliza uma estrutura limpa onde um container principal abriga os itens da galeria.

### Elementos Principais

* **index.html**: Contém a marcação dos elementos e o mapeamento das imagens.
* **style.css**: Arquivo externo responsável pela animação e efeito 3D (deve ser criado separadamente).
